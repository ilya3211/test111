== Description ==

Plugin helps to create expert content on your site.

== Installation ==

1. Upload `expert-review` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 1.2.0 - 2020-01-28 =
Полностью переделан шорткод expert_review, теперь он хранит данные не в разных атрибутах, а в одном. При этом этот
атрибут закодирован для возможности размещения в нем любой информации. Раньше не работали нормально HTML теги, теперь
будут :) Кроме того — добавили превью блока в визуальном редкторе, теперь сам шорткод Вы не видите, вместо него
предварительный просмотр блоков. Ну очень удобно :)

Еще появился шорткод expert_review_likes, но об пока в бета-тесте.
